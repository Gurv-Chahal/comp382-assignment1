Gurveer Chahal & Tudor Roua

Topic 16<br>
Implement a class for an NFA, with a member function that performs the operation of concatenation to another NFA. The NFA class should have constructor to create an executable instance of a particular NFA. The execution should be able to check an input to recognize words in its language.
<br><br>
This assignment was completed using Python, including the random library, which was used to generate random NFAs.
